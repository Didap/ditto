## 1. Visual Theme & Atmosphere

The design follows a **light theme** with a clean, light background (`#f4f4ed`). The primary accent color is a **light lime** (`#d2ff00`) used for interactive elements and brand moments.

Typography is built on **Mona Sans Variable** for headings and **Brier** for body text, creating a balanced dual-family system.

**Key Characteristics:**
- Light background (`#f4f4ed`) with dark text (`#535450`)
- Primary accent: `#d2ff00`
- Font: Mona Sans Variable (headings) / Brier (body)
- Border radius: 7.2px – 14px
- Heading weight: 800, Body weight: 400

## 7. Do's and Don'ts

### Do
- Use `#d2ff00` as the primary interactive/CTA color
- Use `#535450` for headings, `#282c20` for body
- Keep border-radius consistent: 7.2px – 14px
- Use Mona Sans Variable for headings at weight 800
- Apply shadows from the elevation scale for depth
- Use `#dde1d2` for all standard borders

### Don't
- Don't use pure black (#000000) for text — use `#535450`
- Don't mix border-radius sizes inconsistently
- Don't use colors outside the defined palette
- Don't use font weights not present in the type scale
- Don't skip the spacing scale — use the defined values

## 8. Responsive Behavior

### Breakpoints
| Name | Width | Key Changes |
|------|-------|-------------|
| Mobile | <640px | Single column, reduced heading sizes |
| Tablet | 640-1024px | 2-column grids, moderate padding |
| Desktop | 1024-1280px | Full layout |
| Large Desktop | >1280px | Centered content with generous margins |

### Collapsing Strategy
- Navigation: horizontal links → hamburger menu
- Feature cards: multi-column → single column stacked
- Typography scale compresses ~20% on mobile
- Spacing reduces proportionally
