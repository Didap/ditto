## 1. Visual Theme & Atmosphere

The design follows a **dark theme** with a rich, dark background (`#000000`). The primary accent color is a **dark purple** (`#2e0039`) used for interactive elements and brand moments.

Typography is built on **Salesforce-Sans** for headings and **Salesforce-Avant-Garde** for body text, creating a balanced dual-family system.

**Key Characteristics:**
- Dark background (`#000000`) with light text (`#ffffff`)
- Primary accent: `#2e0039`
- Font: Salesforce-Sans (headings) / Salesforce-Avant-Garde (body)
- Border radius: 4px – 12px
- Heading weight: 700, Body weight: 400

## 7. Do's and Don'ts

### Do
- Use `#2e0039` as the primary interactive/CTA color
- Use `#ffffff` for headings, `#f9edff` for body
- Keep border-radius consistent: 4px – 12px
- Use Salesforce-Sans for headings at weight 700
- Apply shadows from the elevation scale for depth
- Use `#f2defe` for all standard borders

### Don't
- Don't use pure white (#ffffff) for text — use `#ffffff`
- Don't mix border-radius sizes inconsistently
- Don't use colors outside the defined palette
- Don't use font weights not present in the type scale
- Don't skip the spacing scale — use the defined values

## 8. Responsive Behavior

### Breakpoints
| Name | Width | Key Changes |
|------|-------|-------------|
| Mobile | <640px | Single column, reduced heading sizes |
| Tablet | 640-1024px | 2-column grids, moderate padding |
| Desktop | 1024-1280px | Full layout |
| Large Desktop | >1280px | Centered content with generous margins |

### Collapsing Strategy
- Navigation: horizontal links → hamburger menu
- Feature cards: multi-column → single column stacked
- Typography scale compresses ~20% on mobile
- Spacing reduces proportionally
